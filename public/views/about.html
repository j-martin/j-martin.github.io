Use keyboard shortcuts

Learn the keyboard shortcuts for every piece of software you use often. Ultimately, you should touch the mouse a little as possible. For example.

In Windows, when in doubt press ALT and type the letter that match what you want to do.

The Windows key can do some cool things. E.g. Windows+1, Windows+X, U, S(and your computer is now sleeping), etc.

On Mac, cmd + space and type what you want.

In SublimeText, CTRL+Por CTRL+SHIFT+P will do everything you'll ever want to do.

In Windows Explorer, Internet Explorer, Chrome, etc, pressing CTRL+L will get you to the address bar prompt.

Good web application (Gmail, vimmium, JIRA, etc.) will give you a list of keyboard shortcuts if you press SHIFT+/ (or ?)
