Some projects

RaspbetrryPi GPIO to ZMQ
Connects the GPIO (Input/Output) of the RaspberryPi to whole world (with ZMQ). In layman terms, it read inputs from anything connected to the RaspberryPi (that cheap little computer) and sends notification by email, sms (text) or pushbullet.
This website
This website is some kind of experiment in web technolgies (from Angular, Node (depending on which server your are accessing) and even configuration management (Puppet and Fabric).

Little tools

paste2file
Small Python utility that copy the clipboard to a file. Can be useful when following a tutorial for example. Coded to work on Windows, *nix and OS X.
font2css
Embeds font files in CSS. Walks through directories (and sub directories), encode TTF and OTF files with base64 and save the result to a CSS file.

Random

dotfiles
My list of dotfiles. Things like vim settings, zsh and such. Mostly only useful to me, but it's always interesting to read other folks config.
photocode
Proof of concept for a web based photo code retrieval system. Someone takes a picture, gives you a ticket with a code and you use that code to get the photo later. Ghetto version of what amusement parks do.
